.cart {
position: relative;
	&:hover {
		.cart-list {
			display: block;
			height: 100px;
		}
	
	}
	.cart-list {
	position: absolute;
	top: 40px;
	right: 0px;
	// 元素高度設定為0
	height: 0px;
	//因為高度為0，文字會跑出來因此隱藏
	overflow: hidden;
	transition: all 0.3s;
	}
}
  • 利用position做定位，讓選單可以出現在下方。
  • 利用高度的變化，滑鼠移入的時候帶入動畫效果，感覺是往下展開。
